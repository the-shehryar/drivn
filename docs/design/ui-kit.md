### 🖼️ User Interface Foundation

## 🎨 Brand Identity

The goal of this UI kit is to maintain a professional, "SaaS-lite" aesthetic across the Next.js admin panel and the React Native client app.

### 🌓 Themes

Currently focusing on **Light Mode** first. Dark mode colors to be defined in Phase 2.

### Deep Petrol #0D2B35 Primary

Wordmark · Navigation · Hero backgrounds · Primary buttons · App shell · Dark surfaces

### Hi-Vis Lime #C8F135 Accent

Logo detail · Key highlights · CTAs on dark · Active nav states · Metric callouts · Brand differentiator

### Muted Petrol #6B7E79 Secondary

Secondary Text · Shadows · Minor Components

### Warm Surface #F2F0EB Base

App background · Card surfaces · Landing page body · Light UI foundation · Off-white warmth

### Neutral Beige #D8D4CC Border

Card borders · Light UI foundation · Component distinction

### Teal Live #00B890 Status · Good

Active vehicle · On-time · Success · Live pulse

### Ember Alert #FF6B35 Status · Alert

Delayed · Exception · Critical · Driver issue

### Caution Amber #F5A623 Status · Warning

Delayed but recoverable · Pending confirmation · Low fuel · Schedule risk

## 📦 Iconography & Assets

Icon Library: Lucide React (Web) & Lucide React Native.

Standard Size: 24x24px for general UI; 16x16px for dense data tables. 12x12 for extra dense spaces.

## 🆎 Typography

### 📱 Mobile App :

Font Family: DM Sans (Primary), Syne (Workmark/Logo).
Heading 1: 18px / SemiBold.
Heading 2: 14px / Medium
Body: 12px / Medium  
Caption: 10px / Medium  
Label / Secondary Text : 8px / Medium

## ⚛️ Atomic Components (Design Specs)

Buttons: Radius: 8px (Rounded).
Focus State: 1px ring of Accent color.

## ⌗ Layout & Spacing Guide

| Layer            | Value | Intent                                                                     |
| ---------------- | ----- | -------------------------------------------------------------------------- |
| Primary Inset    | 20px  | Main screen boundary. All core content starts here.                        |
| Secondary Intent | 16px  | Additional nesting for sub-elements, grouped list items, or indented text. |
| Gutter/Gap       | 10px  | Standard vertical spacing between stacked cards or form elements.          |
